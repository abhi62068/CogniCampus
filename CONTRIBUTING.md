# Contributing to CogniCampus 🎓

First off, thanks for taking the time to contribute! It’s people like you that make CogniCampus a better tool for students everywhere.

## 🚀 How Can I Contribute?

### Reporting Bugs
* Check the [Issues](https://github.com/abhi62068/CogniCampus/issues) tab to see if the bug has already been reported.
* If not, open a new issue. Clearly describe the problem and include steps to reproduce it.

### Suggesting Enhancements
* We are specifically looking for **AIML-related enhancements** (Predictive Bunk Logic, NLP for schedule parsing).
* Open an issue with the tag `enhancement` to start a discussion.

### Pull Requests
1. **Fork** the repository.
2. **Clone** your fork (`git clone https://github.com/your-username/CogniCampus.git`).
3. **Create a branch** for your feature (`git checkout -b feature/AmazingFeature`).
4. **Commit** your changes (`git commit -m 'Add some AmazingFeature'`).
5. **Push** to the branch (`git push origin feature/AmazingFeature`).
6. **Open a Pull Request** against the `main` branch.

## 🛠️ Development Environment
* **Backend:** FastAPI (Python 3.9+)
* **Frontend:** React (Vite)
* **Database:** Supabase (PostgreSQL)

Please ensure your code follows the existing style and includes comments for complex logic.
