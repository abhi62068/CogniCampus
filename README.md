# 🎓 CogniCampus: Intelligent Attendance Strategist

**CogniCampus** is a full-stack academic management ecosystem designed for university students to track, analyze, and optimize their attendance. Built with a focus on the mandatory 75% criteria, it provides real-time insights and predictive "bunk" logic.

---

## ✨ Key Features

* **📊 Advanced Bunk Meter:** Real-time calculation of your safety buffer. Know exactly how many classes you can skip while staying above 75%.
* **📅 Dynamic Engineering Timetable:** Supports 6-day university schedules with customizable period slots.
* **⚡ One-Click Logging:** Fast, persistent attendance marking powered by Supabase.
* **🏥 Academic Calendar Integration:** Automatically handles holidays and exam schedules to keep your dashboard clean.
* **🔐 Secure & Private:** Implements Row-Level Security (RLS) to ensure your data stays yours.
